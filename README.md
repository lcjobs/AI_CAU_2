# 麦小吉 (Mai Xiaoji) - 中国农业大学 AI 智能体网页版

这是一个基于 React 的网页应用，专门用于承载中国农业大学新生助手“麦小吉”的 Coze 智能体。

## 🚨 为什么打开是空白的？(必读)

**不要直接双击 `index.html` 打开！**

因为这个网页使用了专业的 `React` 和 `TypeScript` 技术编写，浏览器本身看不懂这些高级代码。你需要一个“翻译器”（开发服务器）来运行它。

请按照下方的 **【超简单启动教程】** 操作，只需 3 步即可看到画面。

---

## 💻 超简单启动教程 (本地运行)

### 第一步：下载并安装环境
1. **下载代码**：将本项目的所有文件保存在一个文件夹里（比如叫 `cau-xiaoji`）。
2. **安装 Node.js**：去官网 [nodejs.org](https://nodejs.org/) 下载安装包（推荐下载 LTS 版本），一路点击“下一步”安装。
   - *如何验证安装成功？* 按 `Win + R`，输入 `cmd` 回车，输入 `node -v`，如果有版本号跳出来就对了。

### 第三步：启动网页
1. 在你的项目文件夹里打开终端（PowerShell 或 CMD）。
2. 输入 `npm install` 等待安装完成。
3. 输入 `npm run dev`。
4. 浏览器打开显示的网址（通常是 `http://localhost:5173/`）。

## 🌏 【重点】如何实现国内免翻墙访问？

你现在部署在 Vercel 上，后缀是 `.vercel.app`。
**现状**：`*.vercel.app` 域名被中国大陆防火墙（GFW）屏蔽了，所以你需要翻墙才能看。
**解决**：绑定一个**自定义域名**（比如 `maixiaoji.top`）。绑定后，Vercel 会走优选线路，**国内通常可以直接访问，无需备案**。

### 第一步：购买一个便宜的域名
虽然没有“完全免费”且稳定的域名，但我们可以买最便宜的（首年仅需几块钱）。

1. **选择服务商**：
   - **国内**：阿里云、腾讯云（支持支付宝，购买 `.top`, `.xyz` 等后缀，首年约 ¥5-10 元）。*注意：国内购买需要实名认证身份证。*
   - **国外**：[Namesilo](https://www.namesilo.com/), [Spaceship](https://www.spaceship.com/)（支持支付宝，无需实名认证，首年也很便宜）。

2. **购买**：
   - 搜索你想要的名字，比如 `cau-maixiaoji`。
   - 选择 `.top` 或 `.xyz` 这种便宜的后缀。
   - 付款购买。

### 第二步：在 Vercel 绑定域名
1. 登录 [Vercel 控制台](https://vercel.com/dashboard)。
2. 点击你的项目（`ai-assistant-cau...`）。
3. 点击顶部的 **Settings** -> 左侧的 **Domains**。
4. 在输入框里输入你刚买的域名（例如 `www.maixiaoji.top`），点击 **Add**。
5. Vercel 会提示你配置 DNS。它会给你两个值：
   - **Type**: `CNAME`
   - **Value**: `cname.vercel-dns.com`

### 第三步：配置 DNS 解析 (最关键一步)
回到你购买域名的地方（阿里云/腾讯云/Namesilo）的控制台：

1. 找到 **DNS 解析** 或 **域名管理**。
2. 添加一条记录：
   - **记录类型 (Type)**: 选择 `CNAME`
   - **主机记录 (Host/Name)**: 填 `www`
   - **记录值 (Value/Target)**: 填 `cname.vercel-dns.com`
3. 再添加一条（可选，用于不带www访问）：
   - **记录类型**: `A`
   - **主机记录**: 填 `@`
   - **记录值**: 填 `76.76.21.21` (这是 Vercel 的官方 IP)

### 第四步：等待生效
配置好后，等待 5-10 分钟。回到 Vercel 的 Domains 页面，如果两个勾都变成了绿色，恭喜你！
现在发给同学 `www.maixiaoji.top`，他们就可以直接打开了，**不需要翻墙**！

## 🤖 Coze 智能体配置指南

### 1. 身份与逻辑设定 (System Prompt)

请复制以下内容到 Coze 编排页面的 **人设与回复逻辑** 区域：

```markdown
# Role
你叫“麦小吉”，是中国农业大学的专属AI新生助手。你的形象是热情、专业、耐心的学长/学姐。
你的核心任务是根据用户的身份（本科新生 vs 研究生新生）提供差异化的服务。

# Workflow (工作流)
1. **身份确认**：
   - 在对话开始时，如果用户未表明身份，先礼貌询问：“嗨！我是麦小吉。请问你是本科新同学还是研究生新同学呢？不同阶段我有不同的秘籍传授哦！”

2. **分支服务逻辑**：
   - **Scenario A: 用户是本科新生**
     - **保研政策**：解读绩点要求、综测加分项（引用知识库中的《本科生学籍管理规定》）。
     - **升学规划**：提供大一到大四的规划建议（如大一过四六级，大二搞大创）。
     - **奖学金评比**：解释国奖、校奖评选标准。
     - **选课智能**：根据用户兴趣推荐通识课。
     - **真题与日程**：提供考试复习建议和校历关键时间点。

   - **Scenario B: 用户是研究生新生**
     - **文献综述**：根据用户的专业方向（如农业机械化、作物遗传育种等），提供综述的大纲结构和核心关键词。
     - **领域入门**：利用搜索工具查找该领域近3年的高引论文和核心关键词。
     - **科研规划**：提供开题、中期考核、答辩的时间节点建议。
     - **PPT辅助**：为开题答辩或组会汇报生成PPT大纲（Slide by Slide content）。
```
